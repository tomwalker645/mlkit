## 🔬 גרסת Python (מתקדמים)

אם תרצה פלט טקסטואלי, CSV, או גרסת HTML מתקדמת יותר — השתמש בסקריפט Python:

**איך בודקים אם Python כבר מותקן?**

1. לחץ על כפתור **Windows** (⊞) בפינה השמאלית התחתונה של המסך
2. כתוב `PowerShell` ולחץ **Enter**
   > (PowerShell זה כמו "מסך שחור" שאפשר לתת לו פקודות בכתיבה)
3. בחלון שנפתח, כתוב בדיוק:
   ```
   py --version
   ```
   ולחץ **Enter**

- אם כתוב משהו כמו `Python 3.13.2` — מעולה, Python מותקן ✅
- אם כתוב "הפקודה לא זוהתה" — Python לא מותקן, תמשיך לקרוא:

**איך מתקינים Python?**

1. פתח את הדפדפן (Chrome, Firefox, Edge — כל אחד)
2. היכנס לאתר: **https://www.python.org/downloads/**
3. לחץ על הכפתור הצהוב הגדול שכתוב עליו "Download Python"
4. לחץ על הקובץ שהורד
5. ✅ **חשוב מאוד:** בחלון ההתקנה — סמן את הריבוע שכתוב עליו **"Add Python to PATH"** לפני שלוחצים Install
6. לחץ "Install Now" וחכה שיסיים

## 📊 שלב 3 — איך קוראים את התוצאות?

כשיפתח Notepad, תראה משהו כזה:

```
=== Predictor Analysis (v2.0) ===
Target         : בית חגי (Beit Hagai)    ← על איזה יישוב אנחנו מנסים לדעת מראש
Target events  : 47                      ← כמה פעמים הייתה אזעקה בבית חגי בתקופה הזאת
Min volume     : 20                      ← רק יישובים עם לפחות 20 אזעקות
```

### טבלה 1 — "מי מנבא?"

```
#    Settlement                      Total  Precision  Avg Lead  Med Lead  P(sett|tgt)  Hits/Total
--------------------------------------------------------------------------------------------------
1    Ashkelon                           85     72.9%    1m 12s    1m  8s       65.2%       62/85
```

| מה כתוב | מה זה אומר בפשטות |
|---|---|
| `Settlement` | שם היישוב באנגלית |
| `Total` | כמה אזעקות היו בסך הכל ביישוב הזה |
| `Precision` | מתוך כל האזעקות ביישוב הזה — בכמה מהן הייתה אזעקה גם בבית חגי אחר כך? (72.9% = כמעט ב-3 מתוך 4) |
| `Avg Lead` | כמה זמן בממוצע עובר מהאזעקה ביישוב ועד האזעקה בבית חגי (1m 12s = דקה ו-12 שניות) |
| `Med Lead` | זמן החציוני (מדויק יותר מהממוצע כשיש ערכים קיצוניים) |
| `P(sett|tgt)` | מתוך כל האזעקות בבית חגי — בכמה מהן הייתה אזעקה קודם ביישוב הזה? |
| `Hits/Total` | מספר ההצלחות מתוך סך האזעקות ביישוב |

### טבלה 2 — "מי הכי שימושי?"

```
1) Ashkelon  |  precision=72.9%  |  avg_lead=1m 12s  |  total=85  |  score=0.612
```

**זאת הרשימה הכי חשובה** — 3 היישובים שאם תראה אזעקה בהם, עדיף לרוץ למרחב מוגן כי בית חגי תקבל אזעקה בקרוב.

### טבלה 2+ — "ועוד ישובים"

```
4) Sderot  |  precision=68.1%  |  avg_lead=58s  |  total=72  |  score=0.591
5) Netivot  |  precision=65.0%  |  avg_lead=1m 5s  |  total=61  |  score=0.571
```

**טבלה 2+ מציגה את כל שאר הישובים המבצעיים** — הם פחות חזקים מ-3 הראשונים אבל עדיין שימושיים כגיבוי.
